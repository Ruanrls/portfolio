# Sobre

Stellar Fusion é um projeto que tem como objetivo facilitar com que
invesidores realizem seus investimentos de forma mais precisa.

## Desafio

O principal desafio foi lidar com um grande volume de dados e
conseguir processá-los de forma eficiente.

Durante o desenvolvimento desse projeto utilizei web workers para
realizar o processamento dos dados em background. Foi uma experiência
bem interessante que me possibilitou entender mais sobre o fluxo de
trabalho de um web worker, já que foi a primeira vez em que utilizei.
